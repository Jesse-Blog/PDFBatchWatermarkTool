## ✨ 主要功能

*   **🔒 安全隐私**：所有文件处理均在**本地浏览器**完成，文件**不会上传**到任何服务器，绝对保证文档安全。

## 🚀 使用说明

### 第一步：打开工具

双击打开文件夹中的 `PDF批量添加水印.html` 文件，建议使用 **Chrome (谷歌浏览器)** 或 **Edge 浏览器** 以获得最佳体验。

### 第二步：设置水印样式

在左侧面板中配置你的水印：

1.  **水印文本**：输入你想要显示的文字（支持换行）。
2.  **角度与透明度**：拖动滑块调整水印的倾斜角度和深浅。
3.  **字体设置**：
    *   **⚠️ 重要提示**：如果你需要添加**中文水印**，请务必点击 **“上传自定义字体”** 选项，然后选择文件夹中附带的 `微软雅黑粗体.ttf`（或其他你喜欢的 `.ttf` 字体文件）。
    *   *默认的英文字体无法正常显示中文。*
4.  **颜色**：点击颜色块选择喜欢的颜色，或手动输入 RGB 数值。

### 第三步：上传 PDF 文件

将你需要处理的 PDF 文件**直接拖拽**到右侧的虚线框区域，或者点击区域选择文件。支持多选。

### 第四步：开始处理

1.  在“优化选项”中，建议**不要勾选** “优化字体嵌入” （以确保最佳兼容性）。
2.  点击 **“开始添加水印”** 按钮。
3.  工具会自动处理列表中的所有文件，你可以看到实时的进度条。

### 第五步：下载文件

处理完成后，你可以：

*   点击每个文件名旁边的下载链接单独下载。
*   点击底部的 **“批量下载全部文件 (ZIP)”** 按钮，将所有加好水印的 PDF 打包带走。

## 📝 常见问题 (Q&A)

**Q: 为什么中文水印显示成乱码或方框？**
A: 这是因为默认的 PDF 字体不支持中文。请在“字体样式”中选择 **“上传自定义字体”**，并上传一个中文字体文件（如文件夹中的 `微软雅黑粗体.ttf`）。

**Q: 文件处理速度慢吗？**
A: 处理速度取决于你的电脑性能和文件大小。通常情况下，几十个文件只需几秒到几分钟即可完成。

**Q: 我的文件会被上传吗？**
A: **绝对不会**。本工具是纯前端应用，没有后端服务器，所有操作都在你自己的电脑内存中完成，断网也能使用。
